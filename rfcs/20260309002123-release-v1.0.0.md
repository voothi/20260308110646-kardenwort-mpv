## Overview
This release formalizes the transition of the mpv configuration from a simple collection of settings into a sophisticated, script-driven **Language Learning Suite**. The focus was on "Drum" context visualization, intelligent autopause behavior, and a unified, distraction-free OSD system.

## Feature Summary

### 1. Drum Context Mode ('c')
A unique visualization tool that displays the preceding and succeeding subtitle lines around the active dialogue.
- **Dynamic Highlights**: The current line is solid, while context is dimmed/transparent.
- **Smart Stacking**: Automatically calculates gaps when both tracks are at the bottom ('y').
- **ASS Protection**: Disables itself on complex files to avoid rendering artifacts.

### 2. Karaoke-Safe Autopause ('P' / 'K')
The most advanced pause logic for language learners using mpv.
- **End of Phrase**: Pauses only when the sentence is finished.
- **Word by Word**: Uses `{\c}` tags to identify individual word transitions in `.ass` files.
- **Hold-to-Play**: Hold `SPACE` to smoothly bypass all pause points.

### 3. Clean OSD System
Unified status notifications across all study scripts.
- **Position**: Middle-Left (`{\an4}`) to keep the center of the screen clean.
- **Style**: Small 20pt font with a distraction-free 500ms timeout.
- **Universal**: Covers Subtitles, Drum Mode, Autopause, OSC Visibility, and Position toggles.

## Implementation Analytics & Decisions
- **Decision: Keybindings in `input.conf` vs `mp.add_key_binding`**: 
  We shifted all persistent hotkeys to `input.conf`. This solved the issue of key-overwrite bugs between English/Russian layouts and improved the reliably of `complex=true` handlers like the Smart Spacebar.
- **Decision: OSD Styling via `mp.osd_message`**: 
  While `mp.commandv("show-text", ...)` is more modern, it failed to reliably parse ASS tags on the user's environment. We chose the more robust `osd-ass-cc/0` toggle method for maximum compatibility.
- **Decision: 500ms Duration**: 
  The default 2-second timeout was found to be too sluggish for rapid study. 0.5s provides the perfect confirmation without lingering over the text.
