# RFC: Release v1.26.2 (Ultra-Minimalist Search HUD)

**ZID**: 20260312204411
**Date**: 2026-03-12
**Status**: STAGED

## Goal
Finalize the "Black & Bold" search aesthetic by removing all non-textual markers and colors. The objective is to achieve a purely minimalist interface where only the bolded search terms indicate relevance and selection is implicit through navigation.

## Implementation Details

### 1. Marker Removal
- **Decision**: Removed the `> ` and `  ` padding from the search result rendering loop.
- **Rationale**: Eliminates visual noise. The user's focus remains entirely on the subtitle text itself.

### 2. Pure Bold Highlighting
- **Logic**: Character matches continue to use the **Bold** (`{\b1}`) tag. 
- **Consistency**: All results use `text_color` (black/dark grey) against the beige background, ensuring maximum contrast and professional feel.
