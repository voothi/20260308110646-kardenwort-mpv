# RFC: 20260311044229

## Title: Release v1.2.20 - Regression Audit & Release Documentation

### Overview
This release formalizes the comprehensive regression audit between commits `abf23f4` and `27c1e76`, confirming zero regressions across the 18-commit Mouse Selection feature set. It also packages the full release documentation (RFC, release notes, README updates) for the v1.2.18 feature drop.

### Implementation Details

#### Regression Audit
- Performed a full hunk-by-hunk review of the diff (+398/-46 lines, 10 diff hunks) between the pre-feature baseline (`abf23f4`) and the final feature commit (`27c1e76`).
- Confirmed that **all existing functions** (`cmd_dw_copy`, `cmd_dw_word_move`, `cmd_dw_line_move`, `cmd_dw_scroll`, `cmd_toggle_drum`, `draw_drum`, `tick_dw`, `tick_autopause`, `master_tick`, `cmd_smart_space`, `cmd_toggle_sub_vis`, `cmd_cycle_sec_pos`) are completely untouched.
- Verified that the `draw_dw` refactoring preserved functionally identical selection highlighting logic (same `has_selection`, `p1_l/p1_w/p2_l/p2_w` comparisons, same `goto next_word` pattern).
- Validated that all new code is purely additive or precisely scoped to known feature areas.

#### Documentation Updates
- **RFC**: Created `docs/rfcs/20260311023622-release-v1.2.18.md` documenting all technical decisions (layout engine, hit-testing, OS conflict resolution, hardware-accelerated dragging).
- **Release Notes**: Added v1.2.18 "Advanced Mouse Selection" section to `release-notes.md` covering Mouse Selection, UI Conflict Resolution, and Synchronized Scrolling.
- **README**: Bumped version badge to v1.2.18. Expanded "Static Reading Mode" section with Advanced Mouse Selection and Actionable Text (double-click seek). Added `LMB (Drag)`, `LMB (Double)`, and `Ctrl+UP/DN` to the keybindings table.

### Analytics
This is a documentation-only release with no code changes to `lls_core.lua`.
