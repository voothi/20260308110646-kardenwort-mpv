# RFC: Release v1.25.2 (Drum Window Visibility Enhancement)

**ZID**: 20260312195256
**Date**: 2026-03-12
**Status**: STAGED

## Goal
Improve the legibility of the "Active" subtitle line in the Drum Window (Static Reading Mode). The previous "Navy" color had insufficient contrast against the parchment/beige background.

## Implementation Details

### 1. Color Value Adjustment
- **Variable**: `Options.dw_active_color` (defined in `lls_core.lua`).
- **Old Value**: `800000` (Navy - Dark Blue in BGR).
- **New Value**: `D02020` (Bright High-Contrast Blue in BGR).
- **Rationale**: Increasing the blue luminance while maintaining some saturation ensures the active line "pops" without vibrating against the `A9C5D4` background.

### 2. Global Consistency
- **Decision**: Only the Drum Window color was changed. The "Drum Mode" (overlay) active color remains `FFFFFF` (White) as it renders against the video content, where white with black borders provides the best contrast.
