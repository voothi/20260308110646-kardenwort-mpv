# RFC - v1.2.12 (Dual Subtitle Positional Control & Layout Agnosticism)

**Date**: 2026-03-10
**Request ZID**: 20260310141127
**Status**: STAGING

## Executive Summary
This release introduces fine-grained manual control over the positioning of both primary and secondary subtitles. This solve the issue of overlapping multi-line phrasal subtitles by allowing users to adjust their vertical position on-the-fly. Additionally, the standard mpv positioning keys are now layout-agnostic, working seamlessly across English and Russian keyboard layouts.

## Implementation Details

### 1. Independent Secondary Subtitle Positioning
The property `secondary-sub-pos` is now exposed via new keybindings (`Shift+R` and `Shift+T`). This allows the secondary track (translation) to be moved independently of the primary track.
- **Key Binding**: `R` (Up) / `T` (Down)
- **Russian Layout**: `К` / `Е`

### 2. Layout-Agnostic Primary Positioning
Standard mpv positioning keys (`r` and `t` for `sub-pos`) have been explicitly mapped to their Russian counterparts (`к` and `е`). This ensures that users don't have to switch layouts to nudge their subtitles during playback.

### 3. Synergistic Workflow with Drum Mode
The manual positioning persists through "Drum Mode" toggles. While "Drum Mode" has its own internal logic for vertical stacking, users can now pre-adjust the base positions of both tracks in normal mode. This ensures that when the "Drum" is activated, the context lines are rendered exactly where the user needs them based on their custom vertical alignment.

### 4. Documentation Strategy
The "flexibility" and "layout-agnosticism" of the positioning solution are now highlighted in the project documentation as core advantages for multi-language learners.

## Proposed Changes
- **input.conf**: Added layout-agnostic bindings for `sub-pos` (r/t) and `secondary-sub-pos` (R/T).
- **mpv.conf**: Added detailed comments explaining the manual override capabilities.
- **README.md**: Added a "Positional Flexibility" section to features.
- **release-notes.md**: Documented v1.2.12 changes.
