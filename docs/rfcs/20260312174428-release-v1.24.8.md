# RFC: Release v1.24.8 (Stability & Search Selection)

**ZID**: 20260312174428
**Date**: 2026-03-12
**Status**: STAGED

## Goal
The primary objective of this release is to finalize the **Search HUD Selection** features while resolving critical stability regressions discovered during the transition from the experimental `843e4f3` branch back to the stable v1.24.0 baseline.

## Implementation Details

### 1. "Really" Fuzzy Search
- **Decision**: Replaced literal substring matching with a character-order fuzzy algorithm.
- **Rationale**: User feedback indicated that search was not "really fuzzy". The new algorithm allows matching non-contiguous characters (e.g., "hl wrd" -> "hello world"), which significantly speeds up vocabulary lookup during immersion.
- **Analytics**: Implemented via `is_fuzzy_match(str, query)` using UTF-8 table comparisons to ensure multi-byte character support.

### 2. Lexical Scope Stability
- **Decision**: Relocated all Drum Window and Search HUD command functions to be defined BEFORE their respective `mp.add_forced_key_binding` calls.
- **Rationale**: Resolved the "disappearing window" crash. Lua's lexical scoping caused late-bound functions to return `nil` if the binding was managed before the function was defined in the script.
- **Analytics**: Moved `cmd_dw_*` and `utf8_to_table` to the top-level UTILS section of `lls_core.lua`.

### 3. UI Layering (Z-Index Enforcement)
- **Decision**: Explicitly initialized OSD overlays with `z` values.
- **Rationale**: Solved the "subtitle overlap" bug where the Search HUD would occasionally render *behind* native subtitles or the Drum Window.
- **Configuration**: 
    - `search_osd.z = 30` (Topmost)
    - `dw_osd.z = 20`
    - `drum_osd.z = 10` (Bottom)

### 4. Precision Timing (ASS Sync)
- **Decision**: Updated `parse_time` to handle 2-digit centisecond fields.
- **Rationale**: Fixed the desynchronization bug in SRT/ASS playback. Standard players often use 3-digit milliseconds; ASS uses 2-digit centiseconds. The parser now correctly multiplies by 10 for centisecond-based formats.
