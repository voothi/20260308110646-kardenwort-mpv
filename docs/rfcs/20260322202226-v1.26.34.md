# RFC: Release v1.26.34 - Universal Navigation Reliability

**ZID**: 20260322202226
**Status**: Staging
**Release**: v1.26.34

## Summary
This release extends the reliable subtitle navigation fix to all player states by making the precise seeking logic available globally via script-bindings.

## Proposed Changes

### 1. Global Navigation Bindings
- **Infrastructure**: Exported the robust `cmd_dw_seek_delta` function as two new script-bindings: `lls-seek_prev` and `lls-seek_next` in `lls_core.lua`.
- **Global Integration**: Updated `input.conf` to use these bindings for the `a`, `d`, `ф`, and `в` keys.
- **Improved Responsiveness**: By bypassing the native `sub-seek` command in "windowless mode", subtitle jumps now respond instantly on the first press even after an autopause.

## Implementation Details

### Analytics & Decisions
- **Unified Logic**: Instead of maintaining separate navigation paths for Windowed vs. Windowless modes, we now use the same high-precision logic (using the internal `Tracks.pri.subs` table) across the entire suite.
- **Architectural Scalability**: Moving to script-bindings for core navigation makes the configuration cleaner and easier to maintain, as `input.conf` remains the single source of truth for key-to-command mapping.

## Verification Results
- Verified immediate response in Windowless mode after autopause.
- Verified that Drum Window navigation remains functional and consistent.
- Confirmed support for both English and Russian keyboard layouts.
