## ADDED Requirements

### Requirement: Exclusion of Obsolete Scripts
The repository SHALL NOT track or include legacy standalone scripts whose functionality has been fully integrated into the FSM core.

#### Scenario: Auditing script directory
- **WHEN** performing a repository audit
- **THEN** obsolete files like `old_copy_sub.lua` SHALL be removed from tracking.

### Requirement: Ignore Python Bytecode
The repository SHALL ignore temporary build artifacts and bytecode generated by Python-based development tools.

#### Scenario: Committing changes
- **WHEN** Python bytecode (`__pycache__`) is generated in the workspace
- **THEN** it SHALL be excluded from the version control staging area via `.gitignore`.
