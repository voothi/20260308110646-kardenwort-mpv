# Proposal: Compile AGENTS.md

## Goal
The goal is to create a centralized `AGENTS.md` file that documents the available agents, their skills, and workflows within this repository. This will serve as a reference for both users and AI assistants.

## Scope
- List all skills found in `.agent/skills`.
- List all workflows found in `.agent/workflows`.
- Provide brief descriptions of each.

## Non-goals
- Implementing new skills or workflows.
