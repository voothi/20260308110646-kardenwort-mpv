## ADDED Requirements

### Requirement: Configurable Search HUD Aesthetics
The system SHALL provide a set of configuration parameters in the `Options` table to allow users to modify the color and bolding of search results and the query input field.

#### Parameters:
- `search_hit_color`, `search_hit_bold`
- `search_sel_color`, `search_sel_bold`
- `search_query_hit_color`, `search_query_hit_bold`

### Requirement: Ultra-Minimalist Default Theme
The default search HUD presentation SHALL follow a minimalist aesthetic, utilizing color-based selection rather than legacy text markers or bullets.
