## ADDED Requirements

### Requirement: Unified Acquisition Terminology
The project SHALL use the terms **"Language Acquisition"** and **"Immersion"** instead of "Language Learning" or "Study" across all user-facing documentation, metadata, and script comments.

#### Scenario: Auditing documentation
- **WHEN** the user reads the `README.md` or `mpv.conf`
- **THEN** they SHALL find the suite described as a "Language Acquisition Suite" focusing on "Immersion".

### Requirement: Consumption-Focused Mission Statement
The project documentation SHALL explicitly define its primary goal as the **convenient consumption** of dual-subtitle material.

#### Scenario: Reading project goals
- **WHEN** the user inspects the project goals in `README.md`
- **THEN** they SHALL see a focus on the immersion experience and content consumption rather than data processing.
