## ADDED Requirements

### Requirement: Dual-Subtitle Consumption Goals
The system SHALL provide specific documentation for the handling of original and translated subtitle tracks within an acquisition workflow.

#### Scenario: Workflow discovery
- **WHEN** the user reads the "Acquisition Workflow" section
- **THEN** they SHALL find clear instructions on how to toggle and reposition dual-tracks for optimal consumption.

### Requirement: YouTube Context-Loss Mitigation
The project SHALL document how its specialized features (e.g., Drum Mode) mitigate context loss in low-quality subtitle environments like YouTube.

#### Scenario: Handling YouTube subtitles
- **WHEN** the user is acquiring language from YouTube
- **THEN** the documentation SHALL explain how to use "Drum Mode" to maintain visual context despite flickering auto-captions.
